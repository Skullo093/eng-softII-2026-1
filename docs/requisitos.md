REQUISITOS FUNCIONAIS:

| RF01 | O sistema deve permitir ao usuário cadastrar e autenticar sua conta. |
| RF02 | O sistema deve permitir ao usuário registrar receitas financeiras. |
| RF03 | O sistema deve permitir ao usuário registrar despesas financeiras. |
| RF04 | O sistema deve permitir categorizar receitas e despesas. |
| RF05 | O sistema deve exibir o saldo atual consolidado do usuário. |
| RF06 | O sistema deve apresentar histórico de transações financeiras. |
| RF07 | O sistema deve permitir editar e excluir transações registradas. |
| RF08 | O sistema deve gerar relatórios financeiros (mensais, semanais, anuais). |
| RF09 | O sistema deve exibir gráficos de receitas e despesas. |
| RF10 | O sistema deve permitir definir orçamentos por categoria. |
| RF11 | O sistema deve alertar o usuário quando o orçamento for excedido. |
| RF12 | O sistema deve permitir configurar lembretes de contas a pagar. |
| RF13 | O sistema deve enviar notificações sobre vencimento de contas. |
| RF14 | O sistema deve permitir o gerenciamento de múltiplas contas (ex: corrente, poupança). |
| RF15 | O sistema deve permitir exportar dados financeiros (ex: CSV, PDF). |
| RF16 | O sistema deve permitir importar dados financeiros de fontes externas. |
| RF17 | O sistema deve permitir definir metas financeiras (ex: economia). |
| RF18 | O sistema deve acompanhar o progresso das metas financeiras.
