| Código | Descrição |
|--------|----------|
| RF01 | O sistema deve permitir ao usuário cadastrar e autenticar sua conta. |
| RF02 | O sistema deve permitir ao usuário selecionar criptomoedas para monitoramento. |
| RF03 | O sistema deve exibir a cotação atual das criptomoedas selecionadas em tempo real ou quase real. |
| RF04 | O sistema deve permitir ao usuário definir limites de variação de preço para alertas. |
| RF05 | O sistema deve monitorar continuamente as variações de preço das criptomoedas selecionadas. |
| RF06 | O sistema deve enviar notificações ao usuário quando ocorrerem variações significativas de preço conforme limites definidos. |
| RF07 | O sistema deve notificar o usuário sobre eventos relevantes relacionados às criptomoedas monitoradas. |
| RF08 | O sistema deve apresentar notícias relevantes associadas às criptomoedas selecionadas. |
| RF09 | O sistema deve identificar e notificar eventos como regulamentações, falhas de segurança e mudanças de mercado. |
| RF10 | O sistema deve utilizar dados históricos para analisar tendências de mercado. |
| RF11 | O sistema deve calcular a probabilidade de ocorrência de eventos futuros com base em dados históricos. |
| RF12 | O sistema deve alertar o usuário quando a probabilidade de eventos futuros relevantes for alta. |
| RF13 | O sistema deve permitir ao usuário visualizar histórico de preços das criptomoedas. |
| RF14 | O sistema deve permitir ao usuário gerenciar (adicionar/remover) criptomoedas monitoradas. |
| RF15 | O sistema deve permitir configurar preferências de notificações (tipo, frequência, prioridade). |
| RF16 | O sistema deve registrar histórico de notificações enviadas ao usuário. |
